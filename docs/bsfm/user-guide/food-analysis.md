# 🔍 Análise de Alimentos

Guia detalhado para usar o sistema inteligente de análise e registro de alimentos do BSFM, incluindo a tecnologia de IA para identificação automática.

## 🎯 Visão Geral

O módulo de análise de alimentos do BSFM utiliza:
- **🤖 Inteligência Artificial** para reconhecimento de alimentos
- **📊 Database nutricional** com milhares de itens  
- **⚡ Interface intuitiva** para registro rápido
- **📱 Funcionalidade mobile** para uso em qualquer lugar

## 🍎 Métodos de Análise

### **1. Busca Tradicional**

```markdown
🔍 **Como funciona:**
- Digite o nome do alimento
- Sistema sugere matches em tempo real
- Selecione a opção correta

🎯 **Melhor para:**
- Alimentos conhecidos e específicos
- Ingredientes individuais
- Itens da database principal

💡 **Dica:** Use nomes em português para melhores resultados
```

### **2. Análise por Descrição (IA)**

```markdown
🤖 **Como funciona:** 
- Descreva a refeição em texto livre
- IA identifica alimentos e quantidades
- Confirme ou ajuste as sugestões

🎯 **Melhor para:**
- Refeições complexas e preparadas
- Pratos com múltiplos ingredientes  
- Quando não sabe o nome exato

💡 **Exemplo:** "arroz feijão bife salada"
```

### **3. Favoritos e Recentes**

```markdown
⭐ **Como funciona:**
- Acesse alimentos salvos como favoritos
- Reutilize refeições registradas recentemente
- Configure combinações frequentes

🎯 **Melhor para:**
- Refeições do dia a dia
- Lanches e cafés da manhã rotineiros
- Economizar tempo no registro
```

## 📝 Guia Passo a Passo

### **Passo 1: Acessar o Analisador**

1. No dashboard, clique em **"+ Adicionar Refeição"**
2. Selecione o tipo de refeição:
   - 🍳 Café da manhã
   - 🍲 Almoço  
   - 🍽️ Jantar
   - 🍎 Lanches
   - 💧 Água/Outros

### **Passo 2: Escolher Método de Análise**

#### **Opção A - Busca por Nome**
```markdown
1. Clique na barra de busca
2. Digite o nome do alimento (ex: "maçã")
3. Selecione da lista de sugestões
4. Ajuste quantidade e unidade
5. Clique em "Adicionar"
```

#### **Opção B - Descrição com IA**
```markdown
1. Clique em "Descrever Refeição"
2. Digite em português (ex: "frango grelhado com legumes")
3. Revise as sugestões da IA
4. Ajuste quantidades se necessário  
5. Confirme para adicionar todos
```

#### **Opção C - Favoritos/Recentes**
```markdown
1. Clique em "Favoritos" ou "Recentes"
2. Selecione o alimento desejado
3. Ajuste quantidade se necessário
4. Clique em "Reutilizar"
```

### **Passo 3: Revisar e Confirmar**

```markdown
📋 **Revise estas informações:**
- ✅ Alimento correto identificado
- ✅ Quantidade precisa
- ✅ Unidade apropriada (g, ml, un)
- ✅ Reflete a porção realmente consumida

🔄 **Se precisar ajustar:**
- Clique no alimento para editar
- Altere quantidade com os controles +/-  
- Troque unidade se necessário
- Remova itens incorretos
```

## 🍽️ Exemplos Práticos

**Descrição:** "2 fatias de pão integral com 1 ovo cozido"

```markdown

✅ Ovo Cozido - 1 unidade (50g)

📊 Total: ~250 kcal
- Proteína: 15g
- Carbs: 25g  
- Gorduras: 10g
```

## 🎯 Dicas para Melhores Resultados

### **Para Busca por Nome**

```markdown
🎯 **Use nomes específicos:**
- ❌ "carne" → ✅ "contrafilé grelhado"
- ❌ "verdura" → ✅ "brócolis cozido"
- ❌ "suco" → ✅ "suco de laranja natural"

🔍 **Termos de busca eficazes:**
- Nome + preparação (ex: "frango grelhado")
- Nome + cozimento (ex: "arroz cozido")  
- Nome + marca se relevante (ex: "pão integral Wickbold")
```

### **Para Análise com IA**

```markdown
🎯 **Descrições claras:**
- ❌ "comida do almoço" → ✅ "arroz feijão carne salada"
- ❌ "lanche da tarde" → ✅ "iogurte natural com granola"
- ❌ " algo doce" → ✅ "1 fatia de bolo de chocolate"

📝 **Estrutura eficiente:**
1. Ingredientes principais primeiro
2. Quantidades quando souber (ex: "2 colheres de arroz")
3. Preparações importantes (ex: "grelhado", "cozido", "frito")
```

### **Para Quantidades Precisas**

```markdown
📏 **Use medidas caseiras:**
- 🥄 Colheres (sopa, chá)
- 🥣 Xícaras/chávenas  
- 🍶 Copos/mililitros
- ⚖️ Gramas/quilos
- 🍎 Unidades/porções

🎯 **Exemplos práticos:**
- "1 concha de arroz" ≈ 100g
- "1 filé de frango" ≈ 100g  
- "1 fatia de pão" ≈ 25g
- "1 colher de sopa de azeite" ≈ 10ml
```

## 📊 Entendendo as Informações Nutricionais

### **Macronutrientes Principais**

```markdown
🔥 **Calorias (kcal)**
- Energia que o alimento fornece
- Meta diária baseada no seu objetivo

🥩 **Proteínas**
- Construção e reparo muscular
- Importante para saciedade
- Meta: 1.6-2.2g/kg para ativos

🌾 **Carboidratos**  
- Principal fonte de energia
- Simples (açúcares) vs Complexos (fibras)
- Meta: 3-7g/kg dependendo da atividade

🥑 **Gorduras**
- Energia concentrada e funções hormonais
- Saudáveis (insaturadas) vs menos saudáveis (saturadas)
- Meta: 0.8-1.2g/kg
```

### **Micronutrientes Importantes**

```markdown
💧 **Fibras**
- Digestão e saúde intestinal
- Meta: 25-30g/dia

🍬 **Açúcares**
- Limitar adicionados (<50g/dia)
- Natural de frutas é okay

🧂 **Sódio**
- Limitar para saúde cardiovascular 
- Meta: <2300mg/dia

💊 **Vitaminas e Minerais**
- Variedade garante consumo adequado
- Diferentes alimentos fornecem diferentes nutrientes
```

## 🔧 Solução de Problemas

### **Problemas Comuns**

```markdown
🤔 **IA não identificou corretamente:**
- Tente descrever de forma mais específica
- Separe ingredientes com vírgulas
- Use termos mais comuns

🤔 **Alimento não encontrado na busca:**
- Verifique ortografia
- Tente sinônimos ou nomes alternativos
- Use descrição com IA como alternativa

🤔 **Quantidade estimada incorreta:**
- Ajuste manualmente após identificação
- Use medidas caseiras mais precisas
- Considere pesar para referência futura
```

### **Dicas de Precisão**

```markdown
🎯 **Para maior precisão:**
- Pese alimentos quando possível
- Use medidas padronizadas (ex: copo medidor)
- Aprenda a estimar visualmente
- Considere comprar uma balança de cozinha

📊 **Margem de erro típica:**
- Alimentos pesados: ±5%
- Estimativas visuais: ±20%
- Descrições com IA: ±15%
```
