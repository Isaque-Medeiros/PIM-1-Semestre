# 🛠️ Stack Tecnológica

Visão completa das tecnologias, ferramentas e frameworks que utilizo no desenvolvimento de projetos profissionais e acadêmicos.

## 📋 Visão Geral da Stack

### 🎨 **Frontend & Design**
Tecnologias para desenvolvimento de interfaces web modernas e responsivas:

- **HTML5** - Estrutura semântica e acessível
- **CSS3** - Estilização avançada com Flexbox e Grid
- **JavaScript (ES6+)** - Interatividade e lógica client-side
- **Material Design** - Design system para interfaces consistentes

### ⚙️ **Backend & APIs**
Tecnologias server-side para construção de sistemas robustos:

- **Python** - Linguagem principal para automação e APIs
- **Flask/Django** - Frameworks web para aplicações escaláveis  
- **JSON/REST** - APIs RESTful para integração entre sistemas
- **Autenticação JWT** - Segurança e controle de acesso

### 🗄️ **Banco de Dados**
Sistemas de armazenamento e gerenciamento de dados:

- **SQLite** - Banco embutido para protótipos e pequenas aplicações
- **PostgreSQL** - Banco relacional para aplicações production-ready
- **JSON/NoSQL** - Armazenamento de documentos para dados semi-estruturados
- **ORM SQLAlchemy** - Mapeamento objeto-relacional em Python

### ☁️ **Cloud & DevOps**
Plataformas e ferramentas para deployment e infraestrutura:

- **Microsoft Azure** - Serviços cloud para aplicações empresariais
- **Amazon Web Services (AWS)** - Infraestrutura escalável
- **GitHub Actions** - CI/CD para automação de deploys
- **Docker** - Containerização para ambientes consistentes

### 🔌 **Embarcados & IoT**
Tecnologias para desenvolvimento de hardware e sistemas embarcados:

- **Arduino/C** - Programação de microcontroladores
- **Tinkercad** - Simulação de circuitos eletrônicos
- **Sensores & Atuadores** - Integração hardware-software
- **Protocolos Serial** - Comunicação entre dispositivos
