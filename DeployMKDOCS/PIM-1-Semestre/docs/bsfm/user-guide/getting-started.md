# 🚀 Primeiros Passos

Guia passo a passo para configurar sua conta e começar a usar o Brazilian System of Food Metric (BSFM).

## 📋 Pré-requisitos

Antes de começar, verifique se você possui:

- 📧 **Email válido** para cadastro e verificação
- 📱 **Dispositivo com navegador moderno** (Chrome, Firefox, Safari, Edge)
- 🌐 **Conexão com internet** estável
- ⏰ **5-10 minutos** para configuração inicial

## 🎯 Passo a Passo Rápido

### 1. **Acesse o Sistema**
Visite [bsfm-app.railway.app](https://bsfm-app.railway.app) no seu navegador.

### 2. **Crie sua Conta**
Clique em "Cadastrar" e preencha suas informações básicas.

### 3. **Complete seu Perfil**
Adicione detalhes sobre seus objetivos e preferências alimentares.

### 4. **Comece a Registrar**
Adicione seu primeiro alimento e explore as funcionalidades.

## 📝 Cadastro da Conta

### **Formulário de Cadastro**
Preencha as informações necessárias:

```markdown
📋 Dados Obrigatórios:
- Email válido
- Senha segura (mínimo 8 caracteres)
- Confirmação de senha

📋 Dados Opcionais (recomendados):
- Nome completo
- Idade e gênero
- Altura e peso atual
- Objetivos principais
```

### **Requisitos de Senha**
Sua senha deve conter:

- ✅ **Mínimo 8 caracteres**
- ✅ **Pelo menos 1 letra maiúscula** (A-Z)
- ✅ **Pelo menos 1 letra minúscula** (a-z)  
- ✅ **Pelo menos 1 número** (0-9)
- ✅ **Pelo menos 1 caractere especial** (!@#$%^&*)

### **Exemplo de Senha Válida**
```
Senha123!    ✅ Atende todos os requisitos
senhafraca   ❌ Faltam maiúscula, número e especial
12345678     ❌ Faltam letras e especial
Abc123!      ✅ Atende todos os requisitos
```

### **Processo de Verificação**

1. **Email de Confirmação**
   - Você receberá um email com link de verificação
   - Clique no link para ativar sua conta
   - Link expira em 24 horas por segurança

2. **Primeiro Login**
   - Use email e senha cadastrados
   - Sistema lembrará seu dispositivo por 30 dias
   - Opção "Lembrar-me" disponível

### **Nível de Atividade**
Selecione seu nível de atividade física:

| Nível | Descrição | Multiplicador |
|-------|-----------|--------------|
| **Sedentário** | Pouco ou nenhum exercício | 1.2 |
| **Levemente Ativo** | Exercício leve 1-3x/semana | 1.375 |
| **Moderadamente Ativo** | Exercício moderado 3-5x/semana | 1.55 |
| **Muito Ativo** | Exercício intenso 6-7x/semana | 1.725 |
| **Extremamente Ativo** | Atleta profissional | 1.9 |

## 🍎 Adicionando seu Primeiro Alimento

### **Métodos de Adição**

1. **Busca Rápida**
   ```
   🔍 Digite o nome do alimento
   📋 Selecione da lista de sugestões
   ⚖️ Ajuste quantidade e porção
   ✅ Clique em "Adicionar"
   ```

2. **Descrição Detalhada**
   ```
   📝 Descreva a refeição em texto livre
   🤖 IA analisa e sugere alimentos
   ✅ Confirme ou ajuste as sugestões
   ```

3. **Favoritos e Recentes**
   ```
   ⭐ Use alimentos salvos como favoritos
   🔄 Reutilize refeições recentes
   🎯 Configure refeições padrão
   ```

### **Exemplo Prático**

**Adicionando um Café da Manhã:**

1. Clique em "+ Adicionar Refeição"
2. Selecione "Café da Manhã"
3. Pesquise "pão integral"
4. Selecione "Pão Integral" da lista
5. Ajuste quantidade para "2 fatias"
6. Adicione "ovo cozido" - "1 unidade"
7. Clique em "Salvar Refeição"

## 📊 Entendendo os Dados Nutricionais

### **Informações Exibidas**

Cada alimento mostra:

```markdown
🔥 Calorias: Energia total (kcal)
🥩 Proteínas: Construção muscular (g)
🌾 Carboidratos: Energia principal (g)
🥑 Gorduras: Energia concentrada (g)
💧 Fibras: Digestão e saciedade (g)
🍬 Açúcares: Carboidratos simples (g)
```

### **Valores Diários de Referência**

Baseados em dieta de 2000 kcal:

| Nutriente | Valor Diário |
|-----------|--------------|
| **Calorias** | 2000 kcal |
| **Proteínas** | 50-175 g |
| **Carboidratos** | 225-325 g |
| **Gorduras** | 44-78 g |
| **Fibras** | 25-30 g |
| **Açúcares** | < 50 g |

## 🎯 Configurando suas Primeiras Metas

### **Tipos de Metas Disponíveis**

1. **Meta Calórica Diária**
   - Baseada no seu objetivo (perda, ganho, manutenção)
   - Calculada automaticamente ou personalizada

2. **Metas de Macronutrientes**
   - Distribuição entre proteínas, carbs e gorduras
   - Ex: 30% proteína, 40% carbs, 30% gordura

3. **Metas Específicas**
   - Consumo de água (ex: 2L por dia)
   - Ingestão de fibras (ex: 25g por dia)
   - Limite de açúcar (ex: < 50g por dia)

### **Calculadora de Metas**

O BSFM calcula automaticamente:

```markdown
📊 Taxa Metabólica Basal (TMB):
- Calorias que seu corpo queima em repouso

🔥 Gasto Energético Total (GET):
- TMB × Nível de Atividade = Calorias totais

🎯 Meta Final:
- Perda de peso: GET - 500 kcal
- Ganho muscular: GET + 500 kcal  
- Manutenção: GET ± 0 kcal
```
