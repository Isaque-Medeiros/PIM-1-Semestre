# 🎯 Metas

Sistema inteligente de definição, acompanhamento e ajuste de metas nutricionais personalizadas no BSFM.

## 🎯 Visão Geral

O módulo de metas do BSFM oferece:
- **🎯 Metas personalizadas** baseadas no seu perfil e objetivos
- **📊 Acompanhamento em tempo real** do progresso
- **🤖 Recomendações inteligentes** baseadas em dados
- **🔄 Sistema adaptativo** que aprende com seu progresso
- **📱 Notificações e lembretes** para manter o foco

## 🏃‍♂️ Tipos de Metas Disponíveis

### **1. Metas Calóricas**
```markdown
🔥 Controle de Calorias
- Meta diária baseada no objetivo
- Ajuste automático baseado na atividade
- Diferentes metas por dia da semana

🎯 Objetivos:
- ✅ Perda de peso (déficit calórico)
- ✅ Ganho muscular (superávit calórico)  
- ✅ Manutenção (equilíbrio calórico)
- ✅ Preparação para evento
```

### **2. Metas de Macronutrientes**
```markdown
📊 Distribuição de Macronutrientes
- Porcentagens de proteína, carbs e gordura
- Gramas específicos por dia
- Balanceamento automático

🎯 Perfis Comuns:
- 🥩 Alta Proteína (40%P / 30%C / 30%G)
- 🌾 Equilibrado (30%P / 40%C / 30%G)
- 🥑 Low Carb (35%P / 25%C / 40%G)
- 💪 Atleta (30%P / 50%C / 20%G)
```

### **3. Metas Específicas**
```markdown
⭐ Metas Adicionais:
- 💧 Ingestão de água (ex: 2L por dia)
- 🍎 Consumo de fibras (ex: 25g por dia)
- 🚫 Limite de açúcar (ex: <50g por dia)
- 🧂 Controle de sódio (ex: <2300mg por dia)
- 🥗 Variedade alimentar (ex: 20+ alimentos diferentes)
```

### **4. Metas Comportamentais**
```markdown
📝 Hábitos e Rotinas:
- ⏰ Regularidade de refeições
- 🍽️ Horários consistentes
- 🥗 Preparação de refeições
- 📱 Registro diário consistente
- 💤 Sono e recuperação
```

## 🎯 Configurando suas Metas

### **Processo de Configuração**

```markdown
1️⃣ Acesse "Metas" no menu principal
2️⃣ Selecione "Criar Nova Meta"
3️⃣ Escolha o tipo de meta desejada
4️⃣ Configure parâmetros específicos
5️⃣ Revise e confirme a meta
6️⃣ Comece o acompanhamento
```

### **Calculadora Automática**

O BSFM calcula metas automaticamente baseado em:

```markdown
📋 Dados do Perfil:
- Idade, gênero, altura, peso
- Nível de atividade física
- Objetivo principal (perda/ganho/manutenção)
- Preferências alimentares

🔥 Fórmulas Utilizadas:
- Taxa Metabólica Basal (TMB) - Harris-Benedict
- Gasto Energético Total (GET) - TMB × Nível Atividade
- Meta Calórica - GET ± ajuste para objetivo

📊 Exemplo Prático:
- Homem, 30 anos, 175cm, 70kg, Moderadamente Ativo
- TMB: 1662 kcal
- GET: 1662 × 1.55 = 2576 kcal
- Meta Perda de Peso: 2576 - 500 = 2076 kcal/dia
```

### **Configuração Manual**

Para metas personalizadas:

```markdown
🎯 Meta Calórica Manual:
- Defina calorias exatas por dia
- Diferentes metas por dia da semana
- Meta única ou variável

📊 Macronutrientes Manuais:
- Defina porcentagens exatas
- Especifique gramas por nutriente
- Configure diferentes distribuições

⭐ Metas Específicas:
- Valores numéricos específicos
- Limites máximos/mínimos
- Frequências e periodicidades
```

## 🔄 Sistema Adaptativo

### **Ajustes Automáticos**

O BSFM aprende com seu progresso:

```markdown
🤖 Ajustes Inteligentes:

📉 Se perdendo peso muito rápido:
- Sugere aumento de 100-200 kcal
- Alerta para possível perda muscular

📈 Se ganhando peso não intencional:
- Sugere redução de 100-200 kcal  
- Recomenda revisão de porções

⚖️ Se estagnado por 2+ semanas:
- Propõe ajuste calórico
- Sugere mudança na distribuição
- Recomenda consulta profissional
```

## 🎯 Estratégias de Metas

### **Metas SMART**

```markdown
🎯 Metas SMART no BSFM:

S - Específica (Specific)
- ❌ "Comer melhor" → ✅ "Consumir 25g de fibras diárias"

M - Mensurável (Measurable)  
- ❌ "Beber mais água" → ✅ "Beber 2L de água por dia"

A - Atingível (Achievable)
- ❌ "Perder 10kg em 1 mês" → ✅ "Perder 0.5-1kg por semana"

R - Relevante (Relevant)
- ❌ "Meta genérica" → ✅ "Alinhada com objetivo principal"

T - Temporal (Time-bound)
- ❌ "Em algum momento" → ✅ "Por 12 semanas"
```

## 📊 Análise de Progresso

### **Métricas de Sucesso**

```markdown
📈 Indicadores de Performance:

✅ Taxa de Aderência:
- Porcentagem de dias dentro da meta
- Meta: >80% de aderência

📊 Consistência:
- Variação diária/semanal
- Menor variação = maior consistência

🎯 Progresso Real vs Esperado:
- Peso perdido/ganho vs projeção
- Ajuste estratégia se necessário

💪 Composição Corporal:
- Massa gorda vs massa magra
- Foco em qualidade do progresso
```

## 🔄 Revisão e Ajuste

### **Quando Revisar Metas**

```markdown
⏰ Momentos para Revisão:

📅 Periodicidade Regular:
- Revisão semanal (check-in)
- Revisão mensal (progresso)
- Revisão trimestral (objetivos)

🎯 Baseado em Progresso:
- Após 2 semanas sem progresso
- Quando atingir objetivo atual
- Se circunstâncias mudarem
- Por recomendação profissional

💡 Sinais para Ajuste:
- Fome excessiva constante
- Fadiga ou falta de energia
- Estagnação no progresso
- Mudança no estilo de vida
```

### **Processo de Ajuste**

```markdown
🔄 Como Ajustar Metas:

1. Analise dados de progresso
2. Identifique padrões e desafios
3. Considere fatores contextuais
4. Faça ajustes incrementais
5. Monitore resultados
6. Itere conforme necessário

📊 Ajustes Típicos:
- Calorias: ±100-200 kcal
- Macronutrientes: ±5-10%
- Metas específicas: ±10-20%
- Prazos: estender/encurtar
```

## 🚀 Dicas para Sucesso

### **Melhores Práticas**

```markdown
🎯 Para Maior Sucesso:

1. **Comece com 1-2 Metas**
   - Foco evita sobrecarga
   - Domine antes de adicionar mais

2. **Seja Realista**
   - Metas desafiadoras mas atingíveis
   - Progresso consistente > rápido

3. **Celebre Pequenas Vitórias**
   - Semana consistente
   - Meta específica atingida
   - Hábito novo estabelecido

4. **Use os Lembretes**
   - Notificações mantêm foco
   - Check-ins regulares
   - Revisões periódicas

5. **Ajuste Quando Necessário**
   - Flexibilidade é key
   - Aprenda com os dados
   - Otimize continuamente
```

### **Superando Desafios**

```markdown
🛑 Desafios Comuns:

📉 Dias Fora da Meta:
- Não desanime - foque na média semanal
- Analyze o que aconteceu
- Volte ao plano no dia seguinte

🎯 Falta de Motivação:
- Reveja seu "porquê" inicial
- Celebre progresso já feito
- Conecte com comunidade

📊 Estagnação:
- Reanalise suas metas
- Considere pequenos ajustes
- Busque apoio profissional

💡 Lembre-se: Progresso não é linear!
- Altos e baixos são normais
- Consistência a longo prazo
- Foco no processo, não perfeição
```
