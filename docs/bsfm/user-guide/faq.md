# ❓ Perguntas Frequentes (FAQ)

Respostas para as dúvidas mais comuns sobre o uso do Brazilian System of Food Metric (BSFM).

## 📋 Geral

### **O que é o BSFM?**
O BSFM (Brazilian System of Food Metric) é um sistema inteligente de análise e acompanhamento nutricional que utiliza IA para ajudar usuários a atingirem seus objetivos de saúde através do monitoramento preciso da alimentação.

### **O BSFM é gratuito?**
Sim! O BSFM oferece um plano gratuito com funcionalidades completas para usuários individuais. Planos premium com recursos avançados estarão disponíveis em breve.

### **Preciso ser nutricionista para usar?**
Não! O BSFM foi desenvolvido para ser intuitivo e acessível para todos. A IA ajuda a guiar suas escolhas, mas sempre recomendamos consultar profissionais de saúde para orientações personalizadas.

### **Meus dados são seguros?**
Absolutamente! Segurança e privacidade são nossas prioridades:
- Dados criptografados em trânsito e repouso
- Senhas hasheadas com bcrypt
- Conformidade com LGPD
- Nunca vendemos ou compartilhamos dados pessoais

## 🏃‍♂️ Começando

### **Como crio minha conta?**
1. Acesse [bsfm-app.railway.app](https://bsfm-app.railway.app)
2. Clique em "Cadastrar"
3. Preencha email e senha segura
4. Confirme seu email pelo link enviado
5. Complete seu perfil com informações básicas

### **Esqueci minha senha, e agora?**
1. Clique em "Esqueci minha senha" na página de login
2. Digite seu email cadastrado
3. Siga as instruções no email de recuperação
4. Crie uma nova senha segura

### **Preciso fornecer quais informações?**
Informações básicas para personalização:
- Email e senha (obrigatório)
- Nome, idade, gênero (opcional mas recomendado)
- Altura, peso, objetivos (para cálculos precisos)
- Preferências e restrições alimentares

### **Posso usar em múltiplos dispositivos?**
Sim! Seus dados sincronizam automaticamente entre:
- Navegadores web (Chrome, Firefox, Safari, Edge)
- Dispositivos móveis (em breve app dedicado)
- Diferentes computadores

## 🍎 Alimentos e Database

### **Quantos alimentos tem a database?**
Nossa database contém:
- +5,000 alimentos brasileiros e internacionais
- Dados nutricionais precisos e verificados
- Informações de porções e medidas caseiras
- Atualizações regulares com novos alimentos

### **Os dados nutricionais são confiáveis?**
Sim! Utilizamos fontes confiáveis:
- TACO (Tabela Brasileira de Composição de Alimentos)
- USDA FoodData Central
- Dados de marcas e produtos específicos
- Validação contínua pela comunidade

### **E se um alimento não estiver na database?**
Você pode:
1. Usar a análise por descrição com IA
2. Sugerir adição do alimento
3. Usar alimento similar como referência
4. Criar alimento personalizado temporário

### **Como funciona a análise por IA?**
A IA do BSFM:
- Processa descrições em português natural
- Identifica alimentos e estima quantidades
- Aprende com correções e feedbacks
- Melhora continuamente com mais uso

### **Posso adicionar meus próprios alimentos?**
Sim! Você pode:
- Criar alimentos personalizados
- Definir valores nutricionais
- Salvar como favoritos
- Compartilhar com a comunidade (em breve)

## 📊 Metas e Acompanhamento

### **Como as metas são calculadas?**
Baseado em:
- Suas informações pessoais (idade, peso, altura)
- Nível de atividade física
- Objetivo declarado (perda/ganho/manutenção)
- Fórmulas científicas consagradas

### **Posso ajustar metas manualmente?**
Sim! Total controle:
- Ajuste calorias e macronutrientes
- Configure metas específicas
- Defina diferentes metas por dia
- Personalize completamente

### **O que fazer se não estou atingindo as metas?**
1. Analise seus padrões de consumo
2. Ajuste metas para serem mais realistas
3. Use os insights da IA para melhorias
4. Considere consultar um profissional

### **Como o BSFM ajuda com estagnação?**
O sistema oferece:
- Alertas proativos sobre estagnação
- Sugestões de ajustes baseados em dados
- Insights sobre possíveis causas
- Recomendações personalizadas

## 📱 Uso e Funcionalidades

### **Preciso pesar todos os alimentos?**
Não é obrigatório, mas recomendado para precisão:
- Use balança para maior precisão (±5%)
- Estimativas visuais são aceitáveis (±20%)
- IA ajuda com estimativas inteligentes
- Aprenda medidas caseiras com prática

### **Posso registrar refeições completas?**
Sim! Diferentes métodos:
- Item por item (mais preciso)
- Descrição completa com IA (mais rápido)
- Receitas salvas (para repetições)
- Favoritos e recentes (prático)

### **Como funcionam os lembretes?**
Configurável por você:
- Lembretes de refeições
- Alertas de hidratação
- Notificações de progresso
- Lembretes personalizados

### **Posso usar offline?**
Funcionalidade limitada:
- Registre refeições offline
- Dados sincronizam quando online
- Algumas funcionalidades requerem conexão
- App mobile terá melhor suporte offline

## 🔧 Problemas Técnicos

### **O site não carrega, o que fazer?**
1. Verifique sua conexão com internet
2. Limpe cache do navegador (Ctrl+F5)
3. Tente navegador diferente
4. Verifique se JavaScript está habilitado

### **Dados não estão salvando**
1. Verifique conexão com internet
2. Clique em "Sincronizar" manualmente
3. Recarregue a página (F5)
4. Entre em contato com suporte se persistir

### **Recebi erro inesperado**
1. Anote o código de erro (se houver)
2. Recarregue a página (F5)
3. Tente ação diferente e retorne depois
4. Reporte o erro para nossa equipe

### **Como reportar bugs?**
Por:
- Formulário de contato no site
- GitHub Issues ([link](https://github.com/Isaque-Medeiros/BSFM/issues))
- Email para suporte@bsfm.app
- Comunidade de discussão

## 🎯 Dicas e Melhores Práticas

### **Para melhores resultados**
1. **Seja consistente** - Registre diariamente
2. **Seja preciso** - Pese quando possível
3. **Seja honesto** - Registre tudo, mesmo "escapadas"
4. **Use os insights** - Aprenda com os dados

### **Estimativas visuais práticas**
- 🥄 Colher de sopa: 15ml
- 🥄 Colher de chá: 5ml  
- 🍶 Xícara: 240ml
- 🍎 Fruta média: tamanho de punho
- 🍗 Proteína: tamanho de baralho

### **Quando pesar vs estimar**
- **Pese**: Alimentos calóricos, suplementos
- **Estime**: Vegetais, folhas, alimentos low-cal
- **Aprenda**: Compare estimativas com pesagem real
